# Guion: Lección 7 - Clases y Objetos (Integrante 4)

**[0:00 - 0:30] Intro visual:** "¡Hola! Hoy entramos a la mejor parte de Java: la Programación Orientada a Objetos. Imagina un molde para hacer galletas. El molde no se puede comer, ¿verdad? Pero te sirve para crear muchas galletas. En Java, el molde se llama **Clase** y la galleta que te comes es el **Objeto**."

**[0:30 - 1:15] Explicando la Clase:** *(Señalas en pantalla la clase Mascota)* "Aquí creamos nuestro molde, la clase `Mascota`. Toda mascota tiene características, como `nombre` y `edad`."

**[1:15 - 1:50] El Constructor y el Método:** "Pero, ¿cómo nace nuestra mascota? Para eso usamos el **Constructor**. Es la primera instrucción que se ejecuta al crearla, donde le damos su nombre inicial. Además, las mascotas hacen cosas, ¿no? A esas acciones las llamamos **Métodos**. Aquí tenemos el método `saludar`."

**[1:50 - 2:30] Creando el Objeto:** *(Vas al `main`)* "¡Ahora vamos a usar el molde! Con la palabra mágica `new` creamos un objeto real. Le ponemos de nombre Zeus y le damos 3 años de edad. Al presionar 'Run', llamamos a `miPerro.saludar()` y... ¡mira la consola! Ya tienes a tu primera mascota virtual."
